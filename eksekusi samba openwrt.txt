2. Edit Firewall via putty copas paling bawah: 
   CARI FOLDER /etc/config/firewall

config rule
    option src 'lan'
    option proto 'udp'
    option dest_port '137-138'
    option target 'ACCEPT'
config rule
    option src 'lan'
    option proto 'tcp'
    option dest_port '139'
    option target 'ACCEPT'
config rule
    option src 'lan'
    option proto 'tcp'
    option dest_port '445'
    option target 'ACCEPT'
